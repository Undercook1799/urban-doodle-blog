DNS Profiles for iOS & MacOS

Working on a new project this year that configures the DNS server on your local device to another one. Choosing a filtering DNS server will block unwanted requests - by translating the domain name to the IP address of 0.0.0.0 - because by default, unfiltered DNS will let them all pass. This can increase network load and bandwidth, so by blocking them, it essentially speeds up nature of DNS. This is exciting. I know that there are already a loot of good tooks such as dns.notjakob that allows one to create their own DNS configuration, but why not create another one? Heh.
